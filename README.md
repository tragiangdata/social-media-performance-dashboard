# Social Media Performance Dashboard

Interactive Power BI dashboard for analyzing social media engagement, sentiment trends, hashtag performance, and platform comparison.

---

## Project Overview

This project focuses on analyzing social media performance data using Python and Power BI. The workflow includes data cleaning, preprocessing, data merging, and interactive dashboard visualization to identify engagement patterns and content performance across different social media platforms.

The dashboard provides insights into:
- Engagement trends over time
- Sentiment distribution
- Hashtag performance
- Platform comparison
- Top-performing posts

---

## Tools & Technologies

- Python

- Power BI
- CSV Dataset

## Key Features

- Social media engagement trend analysis

- Interactive dashboard filtering with slicers

## Dashboard Components

The dashboard includes:

### KPI Cards
- Total Posts
- Total Engagement
- Average Engagement

### Visualizations
- Engagement trend by date
- Sentiment distribution chart
- Engagement by hashtag
- Average engagement by platform
- Top-performing post details table

### Interactive Features
- Date range slicer
- Platform filter

---

## Key Insights

- TikTok and YouTube achieved the highest average engagement.
- Positive sentiment posts generally received better interaction.
- Certain hashtags significantly increased engagement performance.
- Engagement trends fluctuated noticeably across different periods.

---

## How to Use

1. Open the `.pbix` file in Power BI Desktop.
2. Refresh the dataset if needed.
3. Use slicers to filter data by:
   - Date
   - Platform
4. Explore engagement and sentiment insights interactively.
